# Maratona de Programação — Exercícios

Este repositório tem como objetivo armazenar e documentar as soluções desenvolvidas para os **exercícios da disciplina de Maratona de Programação**, oferecida pela **Universidade Federal de Itajubá (UNIFEI)**.  

## Descrição

A disciplina de Maratona de Programação visa o aprimoramento das habilidades de raciocínio lógico, resolução de problemas algorítmicos e domínio de estruturas de dados, preparando os estudantes para competições de programação e desafios técnicos.

Este repositório reúne as soluções implementadas em **C/C++**, organizadas por temas ou listas de exercícios, conforme o andamento da disciplina.

## Estrutura do Repositório

A organização dos arquivos segue o padrão abaixo:
```text
├── Lista1/
│ ├── problemaA.cpp
│ ├── problemaB.c
├── Lista2/
│ ├── problema1.c
│ ├── problema2.cpp
```

Cada diretório contém os arquivos fonte referentes aos exercícios de um determinado tema.

## Como Compilar e Executar

Os códigos foram desenvolvidos em **C/C++** e podem ser compilados utilizando o compilador `g++`.

### Compilação
```bash
g++ nome_do_arquivo.cpp -o nome_do_programa
```
### Execução
```bash
./nome_do_programa
```
## Objetivos

-Aplicar conceitos de algoritmos e estruturas de dados.

-Praticar resolução de problemas de forma eficiente.

-Desenvolver experiência para competições de programação.

-Aperfeiçoar a escrita de código limpo e organizado.
